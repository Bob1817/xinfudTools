# 快速操作指南

## 📤 推送代码到远程仓库

由于 SSH 密钥有密码保护，请手动执行以下命令：

```bash
cd /Users/shibo/Documents/trae_projects/xinfudTools

# 推送到 Gitee
git push gitee main

# 推送到 GitHub  
git push github main
```

输入你的 SSH 密钥密码即可完成推送。

## 📦 创建 Release

### Gitee Release

1. 访问: https://gitee.com/shibo8817/xinfud-tools/releases/new
2. 填写信息:
   - **Tag 版本**: `v1.0.0`
   - **标题**: `v1.0.0 - 首次发布`
   - **内容**: 复制 `RELEASE_NOTES_v1.0.0.md` 的内容
3. 上传附件:
   - 点击"添加附件"
   - 选择 `dist/HR数据处理工具集-v1.0.0-mac.dmg` (478MB)
4. 点击"发布"

### GitHub Release

1. 访问: https://github.com/Bob1817/xinfudTools/releases/new
2. 填写信息:
   - **Tag version**: `v1.0.0`
   - **Release title**: `v1.0.0 - 首次发布`
   - **描述**: 复制 `RELEASE_NOTES_v1.0.0.md` 的内容
3. 上传附件:
   - 拖拽或点击上传
   - 选择 `dist/HR数据处理工具集-v1.0.0-mac.dmg` (478MB)
4. 点击"Publish release"

## 🪟 Windows 安装包构建

需要在 Windows 机器上执行：

```cmd
cd C:\path\to\xinfudTools

# 运行打包脚本
build_exe.bat
```

**要求**:
- Python 3.10+
- Windows 10/11 (64位)

**可选** (创建安装程序):
- 安装 Inno Setup 6: https://jrsoftware.org/isdl.php

脚本会自动生成:
- `dist/HR数据处理工具集-v1.0.0-windows.exe`
