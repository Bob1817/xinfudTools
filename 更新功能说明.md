# 自动更新功能使用说明

## 功能概述

HR 数据处理工具集现已支持自动更新功能，您可以：
- ✅ 启动时自动检查更新
- ✅ 手动检查更新
- ✅ 查看版本信息
- ✅ 快速前往下载页面

## 使用方法

### 1. 查看当前版本

打开应用后，窗口标题会显示当前版本号，例如：
```
HR 数据处理工具集 v1.0.0
```

### 2. 自动检查更新

每次启动应用时，系统会自动在后台检查是否有新版本。如果有更新：
- 右上角会弹出提示通知
- 点击通知可前往设置页面查看更新详情

### 3. 手动检查更新

**步骤：**
1. 点击左侧导航栏底部的 **"设置"** 按钮
2. 在设置页面中找到 **"软件更新"** 卡片
3. 点击 **"检查更新"** 按钮
4. 系统会连接 Gitee/GitHub 检查最新版本

**结果提示：**
- ✅ **已是最新版本**：显示成功提示
- 📦 **有新版本**：弹出更新对话框，显示版本号和更新日志
- ❌ **检查失败**：显示错误信息（可能是网络问题）

### 4. 下载更新

当检测到新版本时：

**方式一：前往下载页面**
1. 在更新对话框中点击 **"前往下载"**
2. 浏览器会自动打开 Gitee 发布页面
3. 下载最新版本的安装包或源代码

**方式二：通过设置页面**
1. 进入 **设置 > 软件更新**
2. 点击 **"前往下载"** 按钮
3. 打开 Gitee 仓库页面

### 5. 安装更新

由于桌面应用的特性，更新需要手动完成：

1. **下载最新的安装包**（从 Gitee Releases 页面）
2. **关闭当前应用**
3. **运行新的安装包**
4. 按照安装提示完成更新

## 更新源

系统支持两个更新源，优先顺序为：
1. **Gitee**（国内，速度更快）
   - 地址：https://gitee.com/shibo8817/xinfud-tools
2. **GitHub**（备用）
   - 地址：https://github.com/Bob1817/xinfudTools

如果 Gitee 连接失败，系统会自动尝试从 GitHub 获取更新信息。

## 版本管理

### 版本号规则

采用 `主版本.次版本.修订号` 格式，例如：`1.0.0`

- **主版本**：重大功能变更或不兼容更新
- **次版本**：新功能或界面优化
- **修订号**：Bug 修复和小优化

### 查看版本文件

当前版本定义在：
```
core/version.py
```

内容示例：
```python
__version__ = "1.0.0"
__app_name__ = "HR 数据处理工具集"
```

## 常见问题

### Q1: 检查更新时提示"无法连接到更新服务器"

**原因：**
- 网络连接不稳定
- Gitee/GitHub 服务暂时不可用

**解决方案：**
1. 检查网络连接
2. 稍后重试
3. 手动访问 Gitee 页面查看最新版本

### Q2: 更新后数据会丢失吗？

**不会！** 更新只替换程序文件，以下数据完全安全：
- ✅ 历史记录数据库（`data/history.db`）
- ✅ 用户生成的所有申报表
- ✅ 应用配置和偏好设置

### Q3: 如何知道是否有必要更新？

建议查看更新日志，以下情况建议更新：
- 🐛 修复了您遇到的 Bug
- ✨ 提供了您需要的新功能
- 🚀 性能优化和稳定性提升

### Q4: 可以禁用自动检查更新吗？

目前版本会在启动时静默检查更新（仅弹出提示，不会自动下载）。如需完全禁用，请联系开发者。

## 技术实现

### 核心文件

| 文件 | 功能 |
|------|------|
| `core/version.py` | 版本信息定义 |
| `core/updater.py` | 更新检查和下载逻辑 |
| `ui/settings_page.py` | 设置页面（含更新检查） |
| `ui/update_dialog.py` | 更新提示对话框 |
| `ui/main_window.py` | 主窗口（集成自动更新） |

### 工作流程

```
启动应用
   ↓
静默检查更新（后台线程）
   ↓
连接 Gitee/GitHub API
   ↓
比较版本号
   ↓
有新版本？ ──── 是 ───→ 弹出更新提示
   ↓                         ↓
  否                         用户点击"前往下载"
   ↓                         ↓
继续使用                  打开浏览器下载页面
```
