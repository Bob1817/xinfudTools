### 3. 安装包

#### Windows EXE 安装包 ✅

**打包脚本**: `build_exe.bat`

**构建环境要求**:
- Windows 10/11 (64位)
- Python 3.10+
- Inno Setup 6（创建安装程序，可选）

**构建步骤**:
1. 在 Windows 机器上克隆仓库
2. 运行 `build_exe.bat`
3. 等待打包完成
4. 在 `dist/` 目录找到安装包

**生成的文件**:
- `dist/HR数据处理工具集-v1.0.0-windows.exe` (安装程序)

#### macOS

**说明**: 不提供 DMG 安装包

**开发环境运行**:
```bash
cd xinfudTools
pip install -r requirements.txt
python3 main.py
```
