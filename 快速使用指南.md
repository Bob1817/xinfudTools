# 快速使用指南

## 工具 1：发薪表 → 个税申报表

**适用场景**：发薪表已包含完整的个人社保公积金数据

**操作步骤**：
1. 启动应用，点击"工具 1"标签页
2. 点击"选择文件"按钮，选择发薪数据表 Excel
3. 输入所属期（格式：YYYY-MM，如 2025-04）
4. 点击"生成个税申报表"按钮
5. 等待生成完成后，申报表将保存到桌面

---

## 工具 2：社保表 + 发薪表 → 个税申报表

**适用场景**：需要以社保表中的个人社保数据为准生成申报表

**操作步骤**：
1. 启动应用，点击"工具 2"标签页
2. 点击"选择社保表"按钮，选择社保数据表（必须包含【本月发薪明细】sheet）
3. 点击"选择发薪表"按钮，选择发薪数据表
4. 输入所属期（格式：YYYY-MM，如 2025-04）
5. 系统自动显示匹配情况：
   - 社保表数据条数
   - 发薪表数据条数
   - 身份证号匹配数量及比例
   - 仅在发薪表/社保表的数据条数
6. 点击"生成个税申报表"按钮
7. 等待生成完成后，申报表将保存到桌面

**数据合并逻辑**：
- 以身份证号作为匹配关键字段
- 优先使用社保表中的个人养老、医疗、失业、公积金数据
- 若社保表中无匹配记录，则使用发薪表个人部分数据

## 历史记录

**功能**：
- 查看所有生成记录
- 打开生成的申报表文件
- 打开文件所在文件夹
- 删除不需要的记录

**操作步骤**：
1. 点击"历史记录"标签页
2. 点击"刷新"按钮更新列表
3. 选择记录，点击"打开"或"文件夹"按钮

## 常见问题

**Q: 身份证号显示不正确？**
A: 系统会自动清洗身份证号，去除空格、转为大写、去掉末尾的.0

**Q: 匹配率低怎么办？**
A: 检查社保表和发薪表中的身份证号格式是否一致，确保没有多余的空格或特殊字符

**Q: 生成的申报表无法导入税务平台？**
A: 请确认申报表格式是否符合税务平台要求，当前生成的是标准 28 列个税扣缴申报表格式
