## 🔧 安装方法

### macOS
1. 双击 `HR数据处理工具集-v1.0.0-mac.dmg`
2. 将应用拖拽到 Applications 文件夹
3. 从 Launchpad 或 Applications 运行

## 📝 更新日志

### v1.0.0 (2026-04-09)

**首次发布**

#### 核心功能
- 报税合并：发薪表转换为个税申报表
- 支持单表模式（仅发薪表）
- 支持双表合并模式（发薪表 + 社保表）
- 按身份证号码智能匹配和合并

#### UI 优化
- 使用 PyQt6-Fluent-Widgets 现代化界面
- 左侧导航栏：报税合并、历史记录、设置
- 支持拖放文件
- 实时数据预览

#### 自动更新
- 启动时自动检查 Gitee/GitHub 最新版本
- 手动检查更新（设置页面）
- 版本管理和比较

#### 数据处理
- pandas 数据清洗和转换
- 身份证号自动清洗（去空格、去.0、转大写）
- 自动生成 28 列标准个税申报表
- 带格式的 Excel 输出（蓝色表头、交替行背景）
