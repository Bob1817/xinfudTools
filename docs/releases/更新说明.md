## 使用方法

### 启动应用
```bash
./run_mac.sh
```

### 工具 1 使用
1. 点击左侧"工具 1: 单表生成"
2. 点击"添加文件"或拖放发薪表到列表
3. 输入所属期
4. 点击"生成个税申报表"

### 工具 2 使用
1. 点击左侧"工具 2: 多表合并"
2. 左侧添加社保表，右侧添加发薪表
3. 系统自动显示匹配情况
4. 输入所属期
5. 点击"合并并生成个税申报表"
